## Software Intent

ZeroTermux is a non-profit software developed based on Termux

## Language

English

# Key Features

1. Backup and recovery
2. Container switching
3. Source function switching
4. Linux distribution support (Ubuntu, Kali, etc.)
5. Additional enhancements

## Default Package Sources

The default sources can be configured manually. International users are recommended to use the official Termux version at [termux/termux-app](https://github.com/termux/termux-app)

## Recommendation

After gaining familiarity with ZeroTermux, it is strongly recommended that you transition to the official Termux for the best experience and latest updates: [termux/termux-app](https://github.com/termux/termux-app)
